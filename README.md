# Autonomous Human-Following Mobile Robot

RAS 598: Mobile Robotics — Course Project

A vision-based autonomous mobile robot that detects and follows a designated person in indoor environments using a TurtleBot 4 Lite, ROS 2, and YOLOv8-based perception.

Full milestone documentation: [https://hp092.github.io/Autonomous-Human-Following-Mobile-Robot/](https://hp092.github.io/Autonomous-Human-Following-Mobile-Robot/)
